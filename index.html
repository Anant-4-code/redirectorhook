How It Works

One URL. Instant Dialing.

The standard tel: protocol lets browsers open the native phone dialer — on mobile it dials directly, on Windows it launches Phone Link. This service bridges the gap for tools like Google Sheets that don't support tel: in hyperlinks directly.

Google Sheets Tips

How to Type Phone Numbers in Google Sheets

Phone numbers with a leading + sign confuse Sheets — it thinks you're starting a formula. Use one of these two workarounds:

Option A Wrap in double quotes with an equals sign: = "+1 650 253 0000"
Option B Prefix with a single quote: ' +1 650 253 0000 — treats cell as text
Google Sheets · HYPERLINK Formula

Make Phone Numbers Clickable in Google Sheets

Google Sheets blocks tel: in the HYPERLINK function. The workaround is to point your HYPERLINK to this page with the number as a query parameter — it will instantly redirect to tel: and close the tab.

Basic formula (hardcoded number)

=HYPERLINK("https://YOUR_DOMAIN/?number=+16502530000", "Call Google")

Dynamic formula (number in column B, name in column A)

=HYPERLINK("https://YOUR_DOMAIN/?number="&B2, A2)

Replace YOUR_DOMAIN with the actual domain where this page is hosted (e.g. call.example.com). Drag the formula down to apply it to the whole column.

Example layout

| A (Name) | B (Number) | C (Formula) |
|-----------------|-------------------|--------------------------------------------|

Acme Corp | +44 20 7946 0958 | =HYPERLINK("…/?number="&B3, A3)
Google Docs & Slides

Clickable Numbers in Docs & Slides

You don't need a formula here — just the Insert Link feature.

  • Write the phone number (or a label)

    Type the display text you want in your document, e.g. Call Support: +1 650 253 0000

  • Select the text

    Highlight the phone number or label you just typed.

  • Open the Link dialog

    Press Ctrl+K (or Insert → Link).

  • Paste the URL with the number

    Enter: https://YOUR_DOMAIN/?number=+16502530000

  • Click Apply — done!

    The text is now a clickable link that launches the phone dialer.

Web Pages & HTML

Callable Phone Links on Any Website

On websites you fully control, you can use the tel: URI directly — no redirect needed. The browser will hand off to the native dialer or Phone Link.

<!-- Simple tel: link -->
<a href="tel:+16502530000">Call Google Support</a>

<!-- Or use this redirector for analytics / logging -->
<a href="https://YOUR_DOMAIN/?number=+16502530000">Call Google Support</a>
Technical Details

How This Redirector Works

When the URL contains a ?number= query parameter, this page skips the landing UI entirely, fires a hidden tel: anchor click, and then calls window.close() after a short delay so the tab disappears automatically.

Everything runs client-side in this single HTML file — no server, no backend, no tracking.

// Core redirect logic (runs on page load)
const params = new URLSearchParams(location.search);
const number = params.get('number').replace(/[^0-9+]/g, '');

const a = document.createElement('a');
a.href = `tel:${number}`;
a.click(); // opens Phone Link / mobile dialer
setTimeout(() => window.close(), 1800); // auto-close tab
